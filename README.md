# Vehicle CAN Data Analyzer

## About This Project

Over the past six months, I have been learning embedded systems with a focus on C programming, modern C++, Linux, STM32, communication protocols, and embedded software development.

To apply these skills in a practical, industry-oriented project, I started building the **Vehicle CAN Data Analyzer**. This project is designed to simulate how an embedded Linux application receives, processes, and analyzes vehicle sensor data over the CAN (Controller Area Network) protocol using SocketCAN and modern C++.

This repository is also a record of my engineering journey. I am documenting the complete development process—from project planning and implementation to testing and improvements—with regular commits and progress updates. The goal is not only to build a working application but also to demonstrate consistent learning, problem-solving, and professional software development practices.

## Learning Objectives

* Develop modern C++ applications for embedded systems
* Understand Linux SocketCAN and CAN communication
* Design modular and maintainable software
* Process and analyze real-time sensor data
* Practice version control using Git and GitHub
* Document engineering work in a professional manner

I will continue updating this repository as the project evolves, adding new features, improvements, documentation, and testing results throughout the development process.
